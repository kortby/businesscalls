# Route: PUT /storage/{path}

Serves local storage files.

## Technical Details

| Property | Value |
| --- | --- |
| **URI** | `/storage/{path}` |
| **HTTP Methods** | `PUT` |
| **Route Name** | `storage.local.upload` |
| **Controller Action** | `Closure` |
| **Middleware** | *None* |

## How it Works

Maps storage URL requests to local filesystem files.

## How to Use

Request files using their public URL.
