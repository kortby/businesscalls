# Branding Accent & Theme settings

## Overview

Set theme settings or branding brand accent colors.

## How it Works

Configures light, dark, or system mode styles and accent tokens.

## How to Use

Toggle between Light, Dark, or System mode, and select accent colors to brand the portal.

## Technical Details

| Property | Value |
| --- | --- |
| **URL Path** | `/settings/appearance` |
| **HTTP Method** | `GET` |
| **Route Name** | `appearance.edit` |
| **Action Code** | `\Inertia\Controller` |
| **Middleware** | `web`, `auth`, `verified` |
